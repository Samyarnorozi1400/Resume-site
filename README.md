# Portfolio & Resume Website

This is a personal portfolio and resume website template. It is designed to showcase your projects, skills, experience, and contact information in a clean and responsive layout.

## Features

- Fully responsive design using Tailwind CSS
- Hero section with a featured image
- Projects section to showcase your work
- Skills section with icons
- Experience / Achievements timeline
- Contact section with links to email, Telegram, GitHub, etc.
- Smooth scrolling between sections
- Mobile-friendly navigation menu
